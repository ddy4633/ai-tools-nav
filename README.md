# AI Tools Navigator - 发现最好用的AI工具

## 项目简介

AI工具导航站是一个帮助用户发现和选择AI工具的导航网站。涵盖AI写作、图像生成、代码助手、对话AI等领域。

## 技术栈

- **前端**: Next.js 14 + React 18 + TypeScript + Tailwind CSS
- **数据库**: Supabase (PostgreSQL)
- **部署**: Vercel / Dokploy

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build
```

## 环境变量

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
```

## 功能特性

- 🎯 1000+ AI工具收录
- 🏷️ 智能分类和标签
- 🔍 快速搜索
- 📱 响应式设计
- 🌐 SEO优化
- 🚀 快速加载

## 目录结构

```
ai-tools-nav/
```
